## 01 요소 크기 알아내기

### 함수종류

| 종류               | 설명                                       |
| ------------------ | ------------------------------------------ |
| .height()          | 높이                                       |
| .innerHeight()     | 요소의 padding을 포함한 높이               |
| .outerHeight()     | 요소의 border, padding 포함한 높이         |
| .outerHeight(true) | 요소의 margin, border, padding 포함한 높이 |
